# 🎨 ColorIQ Inspect  
## Smart Automotive Colour Quality Inspection Platform

ColorIQ Inspect is an industrial-style web platform built to digitize automotive colour quality inspection using the L*a*b* colour system.

The platform helps suppliers submit actual colour readings remotely and allows managers to review Pass/Fail judgement, Delta E colour difference, 2D tolerance graphs, 3D LAB visualizations, and inspection reports in one place.

## 📌 Problem Statement

In many automotive quality processes, colour inspection is still managed manually using Excel sheets.

This creates problems such as:

- Manual Pass/Fail checking
- Time-consuming report generation
- Difficult supplier-manager coordination
- Higher chance of human error
- Limited visual understanding of colour deviation

ColorIQ Inspect solves this by converting manual Excel-based inspection into a cloud-based digital quality platform.

---

## ✅ Key Features

- 🔐 Supplier and Manager login system  
- 📥 Supplier colour reading submission  
- 🎯 Automatic Pass/Fail judgement  
- 📐 L, A, B tolerance range checking  
- 🌈 Delta L, Delta A, Delta B calculation
